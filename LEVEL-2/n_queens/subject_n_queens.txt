Assignement name : n_queens

Expected files : *.c *.h

Allowed functions : atoi, fprintf, write

-------------------------------------------------------------------------

引数として与えられた n に対する n 女王問題の全解法を出力するプログラムを作成せよ。
負の値でのテストは行わない。
解法の順序は重要ではない。

解法は以下の形式で表示する：
<p1> <p2> <p3> ... \n
ここで pn は各列における女王の位置を示す行インデックスであり、0 から始まる。

For example this should work :
$> ./n_queens 2 | cat -e

$> ./n_queens 4 | cat -e
1 3 0 2$
2 0 3 1$

$> ./n_queens 7 | cat -e
0 2 4 6 1 3 5$
0 3 6 2 5 1 4$
